WEB SEGURANÇA:

Pra começar vamos trazer a tona o que é SSL: a sigla significa Secure Socket Layer e é um mecanismo usado para codificar informações por meio da criptografia. O objetivo do Certificado SSL é proteger dados pessoais ou sigilosos que circulam na web. A segurança do site se tornou cada vez mais necessária, principalmente depois que os navegadores de internet passaram a notificar os usuários sobre a segurança das páginas na web. Com o Certificado SSL o seu endereço na web passa a ser exibido com HTTPS, com a inscrição “Seguro” e a imagem de um cadeado verde.

Validação de Domínio

Entre as opções para o desenvolvedor, a mais simples de ser emitida é a Validação de domínio. Podendo ser emitida a baixo custo ou gratuitamente, ela é a opção padrão e gera a criptografia e mostra os sinais de um site protegido. Alguns serviços de hospedagem o oferecem. saiba mais

Validação de Empresarial

Entre as opções para o desenvolvedor, esta é mais empresarial, de forma a fazer a verificação dos dados da empresa, dando uma segurança a mais ao cliente. Esta cerrtificação é baseada mais especificamente para empresas que solicitarem informações do usuário, logo, além de elaborar uma criptografia mais intensa, pois requisita mais informações do usuário. Alguns serviços de validação oferecem um plano para mantê-lo como um serviço.

Validação de Estendida

Entre as opções para o desenvolvedor, esta é a mais completa e custosa na sua emissão, pois esta é destinada para empresas de renome, na qual é imprescindível o acontecimento de algum vazamento de dados. Além da mais extensa criptografia, é auumentado seu ranqueamento na pesquisa de sites google. Fazendo jus ao nome de quem pode arcar com esse certificado. Alguns serviços de validação oferecem um plano para mantê-lo.
